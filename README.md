## Install build tools
1. Install command
```

./scripts/install.sh

```

2. Init yarn
```

cd your-folder-name
yarn init

Hit enter until the end.

```

3. How to run
```

gulp default

```

## Install gulp-cli
```

npm i -g gulp-cli

```

## Init yarn
```

cd your-folder-name
yarn init

Hit enter until the end.

```

## Install our modules
```

yarn add gulp gulp-sass browser-sync gulp-autoprefixer gulp-clean-css gulp-purgecss del gulp-minify gulp-zip --dev

```

## Install node-sass
```

npm install node-sass --save-dev

```

## How to run
```

gulp default

```

## How to zip
```

gulp zip

```
